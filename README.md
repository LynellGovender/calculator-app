# 🌸 Pastel Phone Calculator

A cute, aesthetic, phone-style calculator built with HTML, CSS, and JavaScript.  
Designed with soft Pinterest-inspired pastel gradients in lavender, light green, and pink tones.

---

## 💖 Features

- 📱 Phone-style layout (portrait design)
- 🌸 Soft pastel gradient UI (lavender, green, pink)
- 🫧 Bubble-style buttons with smooth hover effects
- ✨ Clean, minimal, aesthetic interface
- ➗ Basic calculator functions (+, −, ×, ÷)
- 🎀 Fully responsive centered design

---

## 🎨 Design Style

This project is inspired by:
- Pinterest aesthetic UI
- Soft girl / pastel themes
- Minimal phone calculator layouts

The goal was to create something that feels:
> calm, pretty, and visually pleasing ✨
